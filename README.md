# Expert System - AI-Powered Diagnostic Assistant

A modern, containerized expert system for network and computer hardware troubleshooting. Replaces Google Gemini API with a React-based NLP chatbot integrated with a Python expert engine.

## Features

- **NLP-Powered Chat Interface**: React-based chatbot that extracts technical facts from natural language
- **Rule-Based Expert Engine**: 100+ diagnostic rules with confidence scoring
- **Real-time Context Panel**: Live display of extracted system variables
- **Interactive Diagnosis**: Step-by-step guidance with follow-up questions
- **Containerized Architecture**: Docker Compose for easy deployment
- **Error Resilience**: Graceful fallbacks and comprehensive error handling
- **Advanced Mode**: Toggle for expert users showing rule traces and confidence breakdowns
